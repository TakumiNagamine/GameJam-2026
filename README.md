# バケモノ・ペイ（GameJam_2026）

## 概要
本作品は、ゲームジャムで制作したタワーディフェンスゲームです。  
コントローラーを使用し、迫り寄せてくるエネミーに対してユニットを配置し、敵が地上に到達しないよう防ぐゲームとなっています。

プレイヤーは限られたコストの中でユニットを配置し、状況に応じた戦略的な防衛を行う必要があります。  
シンプルな操作性の中に、配置タイミングや判断による戦略性を持たせることを意識して制作しました。

## 制作背景
ゲームジャムのテーマ「まわす」をもとに企画しました。  
「経済をまわす」という観点から、コストを消費してユニットを配置する仕組みを取り入れています。

テーマとゲーム性を結びつけることを意識し、チームでアイデアを出し合いながら制作しました。

## 開発情報
- 開発期間：約1週間
- 制作人数：5人
- 開発環境：Visual Studio 2022
- 使用言語：C / C++

## 工夫した点

### ■ ユニット管理システム
- 複数ユニットの状態管理を行う仕組みを設計・実装
- 配置条件や挙動を整理し、ゲーム中で安定して制御できる構造を構築
- 可読性と拡張性を意識したデータ設計

### ■ コスト管理とゲームバランス
- ユニット配置に必要なコスト管理機能を実装
- 数値調整を行い、戦略性が生まれるよう設計
- プレイヤーの判断が重要になるゲーム性を意識

## 苦労した点
- 短期間での開発における設計と実装の両立
- チームメンバーへのタスク割り振りと進行管理
- ゲームバランス調整における試行錯誤

## 学んだこと
- チーム開発における進行管理と役割分担の重要性
- 後輩へのサポートや指示出しの難しさと重要性
- 全体設計を意識した実装の必要性
